About Anubhav Chat App

Real-time, modern, and mobile-friendly chat built with Node.js, Socket.io, and a beautiful UI.

Technologies Used

Node.js Express.js Socket.io HTML5 CSS3 (Glassmorphism, Gradients) JavaScript (ES6+) Mobile Responsive Design
  • Node.js & Express.js: Backend server for HTTP and WebSocket connections.
  • Socket.io: Real-time, bidirectional communication between client and server.
  • HTML5 & CSS3: Modern, responsive, and visually appealing UI.
  • JavaScript: Handles all client-side logic and interactivity.

Project Flow

  1. User opens the chat app in a browser (desktop or mobile).
  2. The client connects to the Node.js server using Socket.io.
  3. User enters a username (stored in local storage for persistence).
  4. All chat messages, typing indicators, and user events are sent and received in real-time via WebSockets.
  5. The UI updates instantly for all users, showing messages, typing status, and connection changes.

How It Works

  • Server (index.js): Sets up Express and Socket.io, manages user connections, broadcasts messages and typing events, and serves static files.
  • Client (index.html & script.js):
    • Prompts for username and connects to the server.
    • Displays messages in real-time, with beautiful chat bubbles and typing indicators.
    • Handles connection status, reconnection, and mobile-friendly UI.
  • Mobile Responsive: The UI adapts to all screen sizes, with touch-friendly controls and smooth transitions.
  • Visuals: Uses gradients, glassmorphism, and modern fonts for an eye-catching look.

Key Features

  • Real-time group chat for all connected users
  • Typing indicators and connection status
  • Mobile-first, responsive design
  • Animated, visually appealing UI
  • Automatic reconnection and server switching
  • Browser notifications for new messages
